# Silver Price Prediction - Indian Market

A machine learning-powered web application that predicts silver prices for the Indian market with GST calculations.

## Features
- Real-time silver price fetching
- ML-based price prediction
- Indian market conversion (INR)
- GST calculation (3%)

## API Endpoints
- `/` - Home page with current prices
- `/predict` - Price prediction page
- `/api/predict` - JSON prediction API
- `/api/current-price` - Current price API
